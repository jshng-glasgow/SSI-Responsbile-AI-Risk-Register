SSI Generative AI Risk Register

A community-maintained register of risks associated with the use of AI in Research Software Engineering. Contribute via GitHub.

LLM-generated code can contain subtle bugs or incorrect logic that is incorporated into research software without adequate review, potentially leading to erroneous results in published research. High High Treat AI-generated code with the same review standards as human-written code. Ensure test coverage includes edge cases that LLMs commonly mishandle. Document where AI tools were used in development. Individual RSEs and researchers, research software teams, institutions providing AI coding tools.
